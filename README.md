
# MacroDSL

A text editor with interactive macro creation and execution. Made to be fast.

## Dependencies

 - Zig
 - Raylib
 
On Arch Linux, run
```sh
sudo pacman -S zig raylib
```

## Building native binaries

``` sh
zig build

# to run
zig build run
```
